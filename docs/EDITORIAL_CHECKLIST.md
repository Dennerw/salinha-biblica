# Checklist Editorial — Por Dinâmica

Use este checklist antes de mover uma dinâmica para `status: approved`.

## Bíblico

- [ ] A passagem está citada corretamente (livro, capítulo, versículo)
- [ ] O contexto histórico foi verificado
- [ ] A verdade principal (`mainTruth`) decorre diretamente da passagem
- [ ] A aplicação não excede o que o texto diz
- [ ] Nenhuma promessa condicional do texto é apresentada como absoluta
- [ ] A dinâmica não usa a passagem como mero texto-prova

## Pedagógico

- [ ] A linguagem é adequada à faixa etária indicada
- [ ] A duração é realista para crianças
- [ ] Os materiais são acessíveis e seguros
- [ ] A atividade funciona em sala pequena (quando applicável)
- [ ] Há adaptação para crianças menores (`youngerAdaptation`)
- [ ] Há adaptação para crianças maiores (`olderAdaptation`)
- [ ] Há alternativa sem materiais (`noMaterialsAlternative`)

## Segurança e proteção

- [ ] Compartilhamento pessoal é opcional (explícito na dinâmica)
- [ ] Não há coerção emocional ou espiritual
- [ ] Não há promessa de resultado espiritual específico
- [ ] Atividades físicas têm `safetyNotes` com cuidados de espaço
- [ ] Atividades com alimentos alertam sobre alergias
- [ ] Nenhuma outra tradição cristã é ridicularizada

## Teológico

- [ ] O nível teológico está correto (`essential` / `interpretive` / `denominational`)
- [ ] O risco editorial está correto (`low` / `medium` / `high`)
- [ ] Tags denominacionais estão completas
- [ ] Conteúdo de risco alto tem `requiresPastoralApproval: true`
- [ ] Conteúdo denominacional tem `doctrinalNotice`
- [ ] Há `adaptationInstructions` quando necessário

## Revisão formal

- [ ] Revisor bíblico identificado em `review.biblicalReviewer`
- [ ] Revisor pedagógico identificado em `review.pedagogicalReviewer`
- [ ] Aprovador pastoral identificado em `review.pastoralApprover` (se risco alto)
- [ ] Data de revisão registrada em `review.reviewedAt`
- [ ] Número de versão incrementado se houve alteração significativa
